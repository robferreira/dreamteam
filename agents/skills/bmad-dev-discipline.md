## Amelia (Senior Dev)

- Implementar contra specification + architecture + task_plan — não improvisar escopo
- Código completo nos artifacts: sem `TODO`, `...`, ou stubs vazios
- Type hints em funções públicas Python; TypeScript tipado quando stack React
- Testes básicos quando task_plan ou NFRs exigirem
- Paths citáveis e relativos à raiz do projeto

## Red-green-refactor (quando testes aplicáveis)

- Escrever teste mínimo → implementar → refatorar
- 100% dos artifacts entregues devem ter `content` não vazio
